Notes for Chapter 2 Programming Basics

  • Single Line comments start with //
  • Multi-line comments starting with /* and finishing with */
  • Use typeof for finding out the type of a value. Ex...typeof 'hello'...returns....'string'
  • const is used when the variable will not be reassigned to another value
  • let is used if the variable might be reassigned later in the program
  • Any variable declared outside of a block is said to haveglobal scope. This means it is accessible everywhere in the program.
  • blocks can be used to create a local scope. This means that any variables defined inside a block using the let or const will only be available inside that block and not be accessible outside of that block
  • Constant and variable names can start with any upper or lower-case letter, an underscore, _ , or dollar character, $ . They can also contain numbers, but cannot start with them.Examples....$name, _answer, firstName, last_name, address_line1
  • Variables that start with an underscore generally refer to private properties and methods, so it's best to not follow this convention for your own variable names. The $ character is also used by the popular jQuery library, so using this in your variable names may also cause problems.
  • Constant and variable names are case sensitive
  • 'It's me' // causes an error "It’s me" // this works Another option is to escape the quotation mark 'It\'s me'
  • The backslash is used to escape special characters in strings such as: Single quote marks \' Double quote marks \" End of line \n Carriage return \r Tab \t If you want to actually write a backslash, you need to escape it with another backslash: "This is a backslash \\" ..... "This is a backslash \"
  • we can write a string in all capital letters using the toUpperCase() method
  • the toLowerCase() method, which will write my name in all lower-case letters
  • If you want to know which character is at a certain position, you can use the charAt() method
  • If you want to find where a certain character or substring appears in a string, we can use the indexOf() method
  • If we want the last occurrence of a character or substring, we can use the lastIndexOf() method
  • If all we want to know if a string contains a certain character, then ES2016 provides the useful includes() method
  • To check if a string starts with a certain character, we can use the startsWith() method. Be careful though, it's case-sensitive
  • And we can use the similar endsWith() method to check if a string ends with a particular character
  • The concat() method can be used to concatenate two or more strings together or the +
  • The trim() method will remove any whitespace from the beginning and end of a string
  • the repeat() method that will repeat a string the stated number of times:
  • Template literals are a special types of string that were introduced in ES6. Template literals use the backtick character, ` , to deliminate the string, as shown in the example below:`She said, "It's Me!"` or `Hello!`; or const age = 39; `I will be ${ age + 1 } next year`; << 'I will be 40 next year'
  • The toFixed() method rounds a number to a fixed number of decimal places: Examples: PI.toFixed(3); // only one dot is needed when using constants or variables << "3.142"
  • There is also a useful function called parseInt() that can be used to convert a string representation of a numerical value back into a number
  • if you use parseInt with a decimal, it will remove anything after the decimal point
  • parseFloat() that converts strings into floating point decimal numbers
  • Undefined is the value given to variables that have not been assigned a value.
  • Null means 'no value'. It can be thought of as a placeholder that JavaScript uses to say "there should be a value here, but there isn’t at the moment."

Notes for Chapter 3: Arrays Logic and Loops

  • array constructor function Example: const myArray = new Array(); << []
  • If an element in an array is empty, undefined is returned
  • You don’t even have to use the same types of items inside an array
  • The delete operator will remove an item from an array
  • we have to subtract 1 from the length property. This is because the index starts at 0, so the last item in the array will have an index of one less than the array’s length.
  • if you make the array longer, the extra slots will be filled in with undefined
  • To remove the last item from an array, we can use the pop() method:
  • The shift() method works in a similar way to the pop() method, but this removes thefirstitem in the array
  • The push() method appends a new value to the end of the array.
  • The unshift() method is similar to the push() method, but this appends a new item to thebeginningof the array
  • The concat() method can be used to merge an array with one or more arrays
  • The join() method can be used to turn the array into a string that comprises all the items in the array, separated by commas: Examples: avengers.join(); << 'Captain America, Iron Man, Thor, Hulk, Hawkeye, Black Widow' -------OR------ avengers.join(' & '); << 'Captain America & Iron Man & Thor & Hulk & Hawkeye & Black Widow'
  • A set is a data structure that represents a collection of unique values, so it cannot include any duplicate values
  • A memory leak occurs when a program retains references to values that can no longer be accessed in its memory. This means that memory is being used to store values that are no longer required by the program, effectively wasting system resources.
  • Maps were another data structure introduced in the ES6 specification. They are a convenient way of keeping a list of key and value pairs, and are similar to 'hashes', or 'hash tables' or 'dictionaries' in other programming languages.
